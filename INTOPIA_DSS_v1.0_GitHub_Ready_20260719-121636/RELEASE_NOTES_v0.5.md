# Release Notes — v0.5 Decision Intelligence

## חוויית משתמש

- המותג עודכן ל־EMBA TAU Simulation.
- עיצוב פיננסי חדש בגוני Navy, Gold ו־Teal.
- תפריט צד נפתח רק בלחיצה על כפתור המבורגר ונסגר לאחר בחירת מסך.
- מסכי הנהלה נפרדים: חדר החלטות, כספים, דוחות, מחקר, תרחישים, תמחור, Agent ומדריך.

## נתונים וקליטה

- Q1, Q2 ו־Q3 נקלטים בנפרד ומשמשים תמונת פתיחה ל־Q4.
- נוסף תהליך Upload → Extract → Review → Commit.
- נוספו נתונים פיננסיים לפי מדינה ו־metadata מלא לכל קובץ.
- אסטרטגיה ויעדי Q9 מחולצים לפרופיל מובנה, מוצגים לאישור ומשמשים בבדיקת התאמת ההמלצות.
- נוספו דוח רבעוני, דוח מצטבר ו־snapshot רבעוני.

## קבלת החלטות

- מודל פנימי 50/50 עם כיסוי נתונים וטווח אי־ודאות.
- Rolling Forecast עד Q9.
- המלצות מתועדפות לפי תקציב, נזילות, תמחור, מלאי, קיבולת, טכנולוגיה ומחקר.
- מעבדת תרחישים Low / Base / High עם בדיקת תקציב ורצפת מזומן.
- מחקרי שוק משויכים לצמתי החלטה רלוונטיים בלבד.
- Decision Agent אופציונלי באמצעות OpenAI Responses API וכלי קריאה בלבד.

## ענן ואבטחה

- נוספו טבלאות migration-safe ל־Supabase.
- גיבוי ושחזור כוללים את כל ישויות v0.5.
- המפתחות נשארים בצד השרת; הדפדפן אינו מקבל Supabase Secret או OpenAI API key.
- 12 בדיקות מקומיות עוברות; בדיקת Supabase אמיתית היא opt-in.
